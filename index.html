GHOST LIB

Booting Library Creating Window Managing Tabs UI Elements Notifications

Documentation

Ghost Lib UI is a professional-grade interface library for Roblox, focusing on speed, stability, and minimalist design.

Booting the Library

Use the following loadstring to fetch the latest version of the library from the official repository.

local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/gigatchadeed-design/GhostLIB-luau/main/GhostLibUi.lua"))()

Creating a Window

The main container for your UI elements. It handles the Draggable state and the Open/Close logic.

local Window = Library:CreateWindow("Ghost Hub")

Creating Tabs

Tabs are used to organize your features. You must call .Show() to display a tab by default.

local MainTab = Window:CreateTab("Main")
MainTab.Show()

UI Elements (Widgets)

Element Code Snippet
Button MainTab:CreateButton("Click Me", function() ... end)
Toggle MainTab:CreateToggle("Enable", function(state) ... end)
Slider MainTab:CreateSlider("Value", min, max, default, function(v) ... end)
Paragraph MainTab:CreateParagraph("Title", "Your content here")

Notifications

Send a ghost-themed notification to the user. It will appear at the bottom-right of the screen.

Library:Notification("Alert", "Feature activated!", 5)

Full Example

-- Initialize
local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/gigatchadeed-design/GhostLIB-luau/main/GhostLibUi.lua"))()
local Window = Library:CreateWindow("Ghost")
local Tab = Window:CreateTab("Combat")

-- Add Widget
Tab:CreateToggle("Aimbot", function(bool)
    print("Aimbot is now ", bool)
end)

Tab.Show()
