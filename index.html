Documentation

A minimalist and professional UI library for modern Luau scripts.

Getting Started

Paste this loadstring into your executor to initialize Ghost Lib UI.

local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/gigatchadeed-design/GhostLIB-luau/main/GhostLibUi.lua"))()

Window & Tabs

The core containers for your interface features.

CreateWindow

string Name 
local Window = Library:CreateWindow("Ghost Hub")

CreateTab

string Name 
local Tab1 = Window:CreateTab("Combat")
Tab1.Show() -- Essential to display at start

Widget Gallery

Toggle

string Title, function Callback

Returns a boolean (true/false) whenever the state changes.

Tab1:CreateToggle("Auto Farm", function(state)
    print("Status: ", state)
end)

Slider

string Title, int Min, int Max, int Default, function Callback

Provides a precise numeric value.

Tab1:CreateSlider("WalkSpeed", 16, 200, 16, function(value)
    game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = value
end)

Button

string Title, function Callback 
Tab1:CreateButton("Reset Character", function()
    game.Players.LocalPlayer.Character.Humanoid.Health = 0
end)

Pro Examples

Full Movement Script (Template)

local Library = loadstring(game:HttpGet(""https://pastes.io/raw/local-library-local-tweenservice-game""))()
local Win = Library:CreateWindow("Ghost Exploits")
local Mov = Win:CreateTab("Movement")

-- WalkSpeed Control
Mov:CreateSlider("Speed", 16, 300, 16, function(v)
    game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = v
end)

-- Infinite Jump
local InfJump = false
Mov:CreateToggle("Infinite Jump", function(s) InfJump = s end)
game:GetService("UserInputService").JumpRequest:Connect(function()
    if InfJump then
        game.Players.LocalPlayer.Character:FindFirstChildOfClass('Humanoid'):ChangeState("Jumping")
    end
end)

Mov.Show()
